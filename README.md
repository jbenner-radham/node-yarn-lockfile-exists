yarn-lockfile-exists
====================
Check for a Yarn lockfile in the current working directory.

Install
-------
```sh
$ yarn add yarn-lockfile-exists # Or alternatively: `npm install --save yarn-lockfile-exists`
```

Testing
-------
```sh
$ yarn test # Or alternatively: `npm test`
```
